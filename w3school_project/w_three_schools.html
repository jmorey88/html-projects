How To Add CSS

When a browser reads a style sheet, it will format the HTML document according to the information in the styles sheet.

Three Ways to Insert CSS

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

With an external style sheet, you can change the look of an entire website by changing just one file!

Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

Example

Eternal styles are defined within the <link> element, inside the <head> section of an HTML page:

<!DOCTYPE html>

<html>

<head>

<link rel="stylesheet" href="mystyle.css">

<head>
